Theorem

Given a pairing algebra B (with zero element 0(B) and pairing operation P(BBB)), we can define a notion of join and a notion of inclusion. A subset of the pairing algebra is a DUC-set (downward union closed) if it is downward closed with respect to inclusion, and closed with respect to joins. A relation R between any set C and the pairing algebra is DUC-valued if for any c in C, {y | R(c,y)} is a DUC-set. The theorem CS-DUC-RELNS states that the DUC-valued relations form a closure system, i.e., an arbitrary intersection of DUC-valued relations is a DUC-valued relation.
