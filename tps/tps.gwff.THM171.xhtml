Part of the series THM170-THM172, which provides a nice example of cut. If g commutes with f, and g has a unique fixed point, then f has a fixed point. Similar to THM141. f(II) COMPOSE g(II) = g COMPOSE f IMPLIES.EXISTS x(I) [g x = x AND FORALL z(I).g z = z IMPLIES z = x] IMPLIES EXISTS y(I).f y = y

( f COMPOSE g = g COMPOSE f ) ⇒ ( ∃ x ι . g ( x ) = x ∧ ∀ z ι . ( g ( z ) = z ) ⇒ ( z = x ) ⇒ ∃ y ι . f ( y ) = y )
