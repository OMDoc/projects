Theorem

If there are at least two individuals, not all functions commute with each other. ~[a(I) = b(I)] IMPLIES ~FORALL f(II) FORALL g(II).f COMPOSE g = g COMPOSE f

¬ ( a = b ) ⇒ ¬ ( ∀ f ι → ι , g ι → ι . f COMPOSE g = g COMPOSE f )
