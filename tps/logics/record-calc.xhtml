1. Record Calculus Types

Conceptrecord
The record expression constructor. It takes an even number of arguments, the ones at even position are the record fields and the ones at the odd positions their values.
Conceptselect
The record selector constant. The first argument is a record, the second a record, from which the value for this feature is selected.
Conceptrecord-beta-equal
Record beta equaltiy
Axiom

Selecting a feature from a record gives the appropriate value: record-beta-equal ( . ( record ( ... , ... , F , V , ... , ... ) , F ) , V )

Conceptellipsis
This is a meta-symbol that is used for record ellipsis, it should not be used in formal content.
Concepttuple
The tuple expression constructor.
Conceptproj
The tuple projection function. It takes a number expression N and a tuple as input and returns the N-th component, if the first argument evaluates to N.
