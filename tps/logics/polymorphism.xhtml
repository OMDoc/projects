Polymorphism
Florian Rabe
2008-07-08

This document defines the operators for parametric poplymorphism.

Conceptpolymorphic-term
The polymorphic term constructor. It abstracts over a kinded variable and returns a term. It can be applied to types.
Conceptpolymorphic-type
The polymorphic type constructor. It returns the type of polymorphic terms by abstracting over a typed variable and returning a type.
Conceptpolymorphic-application
The application operator that applies (polymorphic) terms to types.
