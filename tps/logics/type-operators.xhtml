Type Operators
Florian Rabe
2008-07-08

This document defines the operators for type constructors, also called functors.

Conceptfunctor
The functor constructor. It abstracts over a kinded variable and returns a type. It can be applied to types.
Conceptfunctor-kind
The functor kind constructor. It returns the kind of functors by abstracting over a kinded variable and returning a kind.
Conceptfunctor-application
The application operator that applies types (functors) to types.
