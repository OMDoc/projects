Theorem

If some iterate of a function f has a unique fixed point, then f has a fixed point. Example discussed in Resolution in Type Theory, JSL 36 (1971). FORALL f(II).EXISTS g(II) [ITERATE+ f g AND EXISTS1 x(I).g x = x] IMPLIES EXISTS y(I).f y = y

∀ f ι → ι . ∃ g ι → ι . ITERATE+ ( f , g ) ∧ (CAPSIGMA SUP1) ( λ x ι . g ( x ) = x ) ⇒ ∃ y ι . f ( y ) = y
