Theorem

If a set function preserves unions, then it is monotone. The proof requires extensionality, but no set substitutions. FORALL X(OI) FORALL Y(OI) [K(OI(OI)) [X UNION Y] = K X UNION K Y] IMPLIES FORALL X FORALL Y.X SUBSET Y IMPLIES K X SUBSET K Y

∀ X ι → o , Y ι → o . K ( X UNION Y ) = K ( X ) UNION K ( Y ) ⇒ ∀ X ι → o , Y ι → o . X SUBSET Y ⇒ K ( X ) SUBSET K ( Y )
