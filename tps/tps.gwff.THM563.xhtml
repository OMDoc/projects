Theorem

A closure system is a set of sets closed under arbitrary intersections. Let $0$ be any element, $LESS$ be any binary relation and $JOIN$ be any tertiary relation. The next theorem states that the set of sets containing $0$, downward closed with respect to $LESS$ and closed with respect to $JOIN$ is a closure system. This theorem is related to CS-DUC-RELNS -- a generalization in a sense, except that this theorem deals with sets rather than relations.
