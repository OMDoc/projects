Theorem

Every subset of a finite set is finite, using Dana Scott's FINITESET definition of finite (close to FINITE1, but slightly different). FINITESET(O(OA)) C(OA) AND B(OA) SUBSET C IMPLIES FINITESET B

( FINITESET ( C ) ∧ B SUBSET C ) ⇒ FINITESET ( B )
