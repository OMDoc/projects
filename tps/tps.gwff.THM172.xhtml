Part of the series THM170-THM172, which provides a nice example of cut. If g is an iterate of f, and g has a unique fixed point, then f has a fixed point. Closely related to THM15B. ITERATE+ f(II) g(II) IMPLIES.EXISTS x(I) [g x = x AND FORALL z(I).g z = z IMPLIES z = x] IMPLIES EXISTS y(I).f y = y

ITERATE+ ( f , g ) ⇒ ( ∃ x ι . g ( x ) = x ∧ ∀ z ι . ( g ( z ) = z ) ⇒ ( z = x ) ⇒ ∃ y ι . f ( y ) = y )
