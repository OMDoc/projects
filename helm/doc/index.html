An OMDoc interface to Coq via HELM

In this project we develop interface code for the Coq theorem prover. The project code is available by anonymous CVS (path omdoc/projects/helm), and can viewed here.

Components

The interface code consists of four style sheets that take the XML output of Coq produced by the HELM interface and generates OMDoc from this. This is still in a largely preliminary state. In particular, we can only work for OMDoc2.0, since the mutuality in definitions in OMDoc 1.0 is too weak.

What is still lacking?

  • an integration with the HELM stuff, (an interface to uwobo), so that we can actually run and test it
  • merging with the progress in the HELM project.
  • some test CIC cases!
  • more work on OMDoc 2.0 so that it is expressive enough for CIC.
